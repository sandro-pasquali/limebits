Library - selector

The selector library allows a developer to easily run CSS selectors against the DOM and work with the results.

Methods

select
  • String A CSS selector.
Returns an Array of Element objects matching selector.

When used as part of an AXIS chain, the #select method replaces the #store method. This allows you to run methods in the scope of an array of DOM elements. Such as: 

        AXIS
          .select('h2')
          .foreach(function(t) {
            // gives all <h2> elements red text
            t.style.color = 'red';
match
  • Element The Element to check.
  • String A CSS selector.
Returns Boolean true if selector matches element, false if not.

Checks whether a given selector would match a given Element. 

        // Whether first matched element has `h2` tag
        AXIS
          .select('h2')
          .match(AXIS.$[0],'h2');
        
        // true
        console.log(AXIS.$);
