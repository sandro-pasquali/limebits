Library - array

The array library provides a suite of useful methods for manipulating arrays.

Methods

filter
  • Function A comparison function which returns a Boolean.
Returns Array.

Eliminate elements which resolve false against passed comparison function. 

      AXIS
        .scope([1,2,3])
        .filter(function(f) {
          return f < 3
        })
        
      // [2,1]
      console.log(AXIS.$);
foreach
  • Function A function to process each value.
Returns Array.

Replaces each element of an array with the result of current value passed to function. 

      AXIS
        .scope([1,2,3])
        .foreach(function(f) {
          return f+1
        })
        
      // [2,3,4]
      console.log(AXIS.$);
shuffle
  • Boolean Whether to do deep shuffling (multilevel array shuffling).
Returns Array.

Shuffle an array using Fisher-Yates algorithm. Sorts only zero level of array, unless deep parameter is specified. 

      AXIS
        .scope([1,2,3])
        .shuffle()
        
      // [3,1,2] or similar
      console.log(AXIS.$);
clone
Returns Array.

Creates an independent clone of an array.

equalTo
  • Array The array to compare against.
Returns Boolean.

Checks for string equality between two arrays. 

      var b = [1,2,3];
      var c = [true,2,3];
      AXIS
        .scope([1,2,3])
        .equalTo(b).store('first')
        .equalTo(c).store('second');
        
      // true
      // false
      console.log($AXIS.first.$);
      console.log($AXIS.second.$);
last
Returns Mixed.

Get last element of array.

contains
  • Mixed The search value
  • Boolean Do partial search.
Returns Boolean.

Determines if a given value exists in an array. Supports the search of multidimensional arrays. Supports partial string matches (finding 'a' in 'cat'), or strict equivalency (s === val). 

      var b = [1,2,3];
      var c = ['cat','dog'];
      AXIS
        .scope(b)
        .contains(6).store('first')
        .scope(c)
        .contains('a',true).store('second');
        
      // false
      // true
      console.log($AXIS.first.$);
      console.log($AXIS.second.$);
