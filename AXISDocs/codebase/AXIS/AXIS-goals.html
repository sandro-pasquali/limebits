AXIS Goals

Limebits is focused on providing a uniquely powerful environment for developers. As a greater number of people and organizations invest time and money in the creation of web applications the need for useful development tools grows. To help developers, Limebits aims to shorten the development cycle by allowing the copying of complete applications; this is the 'Copy and Extend' advantage of Limebits. In addition, the 'No More Than Necessary' philosophy of Limebits has resulted in the creation of a powerful application development framework, written in Javascript, powered by WebDAV, named 'AXIS', which is open and free and will help you write better applications faster.

Why Limebits Matters

The internet made everyone's computer and software interoperable. The original goal was to make the internet readable and writable. You should be able to copy what you see and modify it. Limebits gives you that power, and more:

Power

You can now easily author your own software.

Reach

You can distribute your software for free.

Collaboration

You can collaborate with everyone to make your (and their) software better.

AXIS + WebDAV -- Copy and Extend

Any application you see on Limebits is copyable. As applications on Limebits are purely javascript-based, you do not have to understand databases or special server-side code to modify those copies. You can in fact write any HTML and Javascript you'd like, and if it will work in a browser it will work in Limebits. But Limebits gives you more: the ability to use the power of WebDAV. WebDAV lets you read, write, copy, move, modify, and set properties on internet resources. The AXIS opens the internet up for you. It also gives you security control and error handling. It gives you a professional framework allowing you to write and share and extend your and other people's code. The AXIS framework helps you share, expand, and improve your work.

AXIS + Extensions -- An Example

Do you like this documentation system? It is an AXIS extension. You can use it too. You can even modify it. Look how easy it is to create: 

  
    
    
    <script type="text/javascript" src="/!lime/root/library/limebits/latest/axis/AXIS.js" 
    extensions="Modules" 
    libraries="selector_lite+array"
    ><script type="text/javascript">
  
      AXIS.Modules.load({
       	provider: 'local',
       	module:   'Docs',
       	version:  '1',
       	options:  {
       	            rootPath: '../../'
       	          }
      });
      
      AXIS.Modules.load({
        provider: 'local',
        module:   'SyntaxHighlighter',
        version:  '1'
      });
            
    </script>
