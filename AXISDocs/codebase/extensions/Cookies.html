AXIS - Cookies

Methods for handling the creation/deletion of browser cookies.

Cookie Methods

create
  • String Cookie name.
  • String Cookie value.
  • Object Further cookie definition information:
    • String path : Cookie path restrictions.
    • Number days : Expiration date.
Returns Boolean indicating success or failure.

Creates a cookie. 

      // Sets cookie for '/myhome/' expiring in 2 days.
      AXIS.Cookies.create('myCookie','shortbread',{
        path: '/myhome/',
        date: 2
      });
update
  • String Cookie name.
  • String Cookie value.
  • Number Expiration date.
Returns Boolean indicating success or failure.

Updates the value of an existing cookie. 

      // Updates 'myCookie' cookie, which now expires in 7 days.
      AXIS.Cookies.update('myCookie','shortbread',7);
read
  • String Name of cookie to read.
Returns Boolean false if no cookie found, or String with cookie value.

Gets the value of a cookie. 

      console.log(AXIS.Cookies.read('myCookie');
      // shortbread
erase
  • String Name of cookie to erase.
Returns Boolean, false if no such cookie found, true if erased.

Erases a cookie. 

      AXIS.Cookies.erase('myCookie');
      console.log(AXIS.Cookies.read('myCookie');
      // false
